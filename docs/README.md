# Angular Enterprise Boilerplate

## Overview
Boilerplate Angular standalone + Tailwind pensado para aplicaciones empresariales reales.

Incluye:
- Auth JWT + Refresh Token
- CSRF
- RBAC (roles + permissions)
- CRUD de usuarios
- UX profesional (loaders, toasts, estados)
- Arquitectura escalable y mantenible

Este proyecto no es un demo, es una base de producto.
