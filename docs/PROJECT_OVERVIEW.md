# Project Overview

## Visión
Este proyecto es un boilerplate frontend enterprise diseñado para integrarse con un backend NestJS robusto.

Objetivos:
- Escalabilidad
- Seguridad
- Claridad arquitectónica
- UX profesional

## Principios
- Separación clara de responsabilidades
- Core mínimo
- Features aislados
- UI reutilizable
- Contrato claro con backend

## Autenticación
- Login / Logout
- Refresh token rotativo
- CSRF
- Forgot / Reset / Change password

## RBAC
- Roles y permissions derivados
- Guards en rutas
- Validaciones en UI y backend

## UX
- Loader inicial
- Loaders por página
- Loaders por acción
- Toasts globales
- Estados 403 / 404 / empty

Este boilerplate está listo para producción.
